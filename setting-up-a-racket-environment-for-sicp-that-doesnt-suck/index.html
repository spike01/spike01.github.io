Setting up a Racket environment for SICP that doesn't suck

14 May 2015 on Brackets. Brackets everywhere

This is for OS X, because I'm too trendy for *nix.

  1. Install Racket: brew install plt-racket. I assume you have brew
  2. Install xrepl, which gives you normal REPLy things like… arrow keys, history, and all the other things you're used to. raco pkg install xrepl. Raco is the Racket package manager. It’ll take a while. Type a instead of Y at the prompt to minimise annoyance. Go make some tea.
  3. Alias racket -i -p neil/sicp -l xrepl to something. I’ve gone with sicp so I can still use normal Racket. You can do whatever the hell you want though.
  4. Wonder why you’re typing so many brackets all the time for the next few weeks.
